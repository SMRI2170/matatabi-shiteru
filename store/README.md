# ストア公開用アセット（また旅している）

Google Play Console にアップロードする画像一式です。

| ファイル | 用途 | サイズ |
|---|---|---|
| `icon_512.png` | アプリアイコン | 512×512 |
| `feature_graphic.png` | フィーチャーグラフィック | 1024×500 |
| `play_1_map.png` | スクリーンショット① 地図・自動チェックイン | 1080×1920 |
| `play_2_stats.png` | スクリーンショット② 訪問サマリー・地方制覇 | 1080×1920 |
| `play_3_badge.png` | スクリーンショット③ チャレンジ・バッジ | 1080×1920 |
| `play_4_level.png` | スクリーンショット④ 県のレベル・訪問日数 | 1080×1920 |
| `play_5_note.png` | スクリーンショット⑤ メモ・写真 | 1080×1920 |

## Play Console での登録先
- ストアの掲載情報 → アプリのアイコン：`icon_512.png`
- ストアの掲載情報 → フィーチャーグラフィック：`feature_graphic.png`
- ストアの掲載情報 → スマートフォンのスクリーンショット：`play_1`〜`play_5`（2〜8枚必須）
